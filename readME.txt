The following modules are currently required to start this project:
    - express (npm install express)
    - mongodb (npm install mongodb)
-------------------------------------------------------------------
To start this project:
    - Run 'node start.js' in terminal
    - Type "localhost:3000" into your browser (chrome is best)
-------------------------------------------------------------------
Remarks
    - Because the data is fetched from mongodb, 
    - it may not show up at first, waiting a few seconds.
